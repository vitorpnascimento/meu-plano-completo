# 💪 Meu Plano Completo

App de acompanhamento de dieta, treino, peso e notas com React + Next.js

## 🚀 Como Rodar Localmente

### 1. Clone o repositório
```bash
git clone https://github.com/vitorpnascimento/meu-plano-completo.git
cd meu-plano-completo
```

### 2. Instale as dependências
```bash
npm install
```

### 3. Rode o servidor de desenvolvimento
```bash
npm run dev
```

O app vai estar disponível em `http://localhost:3000`

## 📋 Funcionalidades

### Aba "Hoje"
- ✅ Marcar refeições (5 refeições, 20 itens)
- 📊 Cálculo automático de calorias e macros em tempo real
- 🎯 Meta diária de 1.900 kcal
- 💾 Checkboxes funcionando perfeitamente
- 📈 Contador de refeições completas (0/5)

### Aba "Peso"
- 📈 Registrar peso com data
- 📋 Histórico completo
- 🎯 Meta de perda de peso
- 📊 Visualização de progresso

### Aba "Cardio"
- ⏱️ Timer de cardio com minutos customizáveis
- ▶️ Pausar/Resetar
- 📅 Plano recomendado (Semanas 1-4)
- 💡 Dicas de intensidade

### Aba "Notas"
- 📝 Adicionar notas rápidas
- 🕐 Timestamp automático
- 📌 Histórico do dia
- 💬 Registre sentimentos e observações
